# Food Delivery App

## Description
This is a simple Food Delivery App for Android devices that allows users to order food.

## Features
- User Registration and Login.
- Menu and Dish Details.
- Ratings and Reviews.
- User-friendly interface with intuitive design.
  
## Database
- Firebase
- SQLite
